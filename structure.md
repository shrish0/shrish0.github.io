# Project Structure

- `src/views/HomeView.vue` -> Main landing page layout, hero content, portfolio sections, and floating chatbot trigger.
- `src/components/ai-chat-panel.vue` -> Reusable AI chatbot panel UI with starter prompts, local conversation state, and portfolio-aware replies.
- `src/components/technical-modal.vue` -> Skill detail modal opened from the technical modules carousel.
- `src/utils/constants.ts` -> Shared static URLs and app-wide constants.
- `public/resume.json` -> Portfolio content source for hero, skills, projects, experience, and contact details.
